Inventory Web App

A web application for managing inventory and stock with a modern design

Project Overview

This Inventory Web App is designed to help businesses manage their inventory and stock efficiently. It features a user-friendly interface built with Laravel and Vue.js, allowing users to easily add, update, and track inventory items. The dashboard provides real-time data and analytics, helping users make informed decisions about their stock levels.

Technology Used

Laravel Vue JS Inertia JS Tailwind CSS MySQL
